# Packages

This page lists framework packages from a consumer perspective.

Package availability can change while the framework is in active development. Check NuGet and the repository for the current published set.

## Stable Packages

| Package | Purpose |
| --- | --- |
| `IOKode.OpinionatedFramework.Ensuring` | Ensure API for preconditions, postconditions, and invariants. |

## Framework Areas

The framework includes packages or projects for areas such as:

- Command execution.
- Emailing.
- Logging.
- Configuration.
- Encryption.
- Password hashing.
- File systems.
- Persistence and query execution.
- Jobs.
- Events.
- ASP.NET Core integrations.
- Source generators.

Use package-specific documentation when a package is published and stable.
